Release Notes — Current Program

  • DNS Lookup fix: DNS prompt now runs on the main thread to avoid Tkinter dialog errors; results stream correctly into the Network Tools output box.
  • Netstat improvement: Netstat button uses numeric output with PID (e.g., netstat -ano) so connections can be traced to processes.
  • Speedtest compatibility: Added a small shim so speedtest runs correctly in the packaged EXE (handles Python3 __builtin__ expectation and missing stdout).
  • Windowed EXE support: Early guard added to ensure sys.stdout / sys.stderr are valid in a windowed build (prevents fileno() errors).
  • Legacy import shim: A tiny __builtin__.py compatibility file is included for libraries expecting the Python-2 module name.
  • No console pop-ups: Subprocess launches (e.g., robocopy) are run with flags that keep Windows command windows hidden when running the GUI EXE.
  • Bundled assets: Logos and images are included in the EXE build so the animated/branding icons display correctly.
