Last Night Automation

“Automate everything. Sleep well. Wake up protected.”

Last Night Automation is your all-in-one Windows technician tool. It runs silently in the background, securing your system, backing up data, and preparing your machine for a new day.

⬇ Download v1.0.3

🚀 Features

  • Auto backup of My Documents to C:\Backups of Documents
  • Windows Defender scan (logs results)
  • Internet connection check with fallback
  • Silent Windows Updates (skips if offline)
  • Windows toast notifications after each task
  • Modular, fast, and terminal-only (no GUI errors)
